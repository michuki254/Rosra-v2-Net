# Rosra-v2-Net

Revenue gap analysis tool for municipalities.

## Overview
RosraApp is a web application that calculates revenue gaps across different sources including:
- Property Tax
- License
- Short Term
- Long Term
- Mixed

The app visualizes the gap data using charts to help municipalities identify potential revenue opportunities.

## Features
- Detailed gap breakdown by revenue type
- Gap analysis visualization with stacked bar charts
- Revenue comparison across different sources
- Calculation of potential revenue estimates
